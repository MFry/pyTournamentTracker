## Quick Start
If you setup the provisions for the vagrant VM very few steps are needed to get up and running.
```
vagrant up
vagrant ssh  
```
In VM
``` bash
source /vagrant/tournament/virtualenv/bin/activate
cd /vagrant/tournamnet/
make -f Makefile
python sample_run.py    
```
